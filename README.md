# Kingdom & Custody

A Fabric mod for Minecraft 1.21.11 featuring a royal armor hierarchy, inventory scanner, and key system.

## Features
- 21 armor tiers from King to Immortal
- 5 custom ores with unique crafting progression
- Inventory Scanner block with wireless lever pairing
- 20 unique keys that emit a 9-second redstone signal
- Custom `/give <player> [immortal_kit]` command

## Armor Tiers
King → High King → Overlord → Grand Duke → Archduke → Prince Royal → Warlord → Champion → Conqueror → Sovereign → Emperor → High Emperor → Grand Emperor → Supreme Ruler → Celestial King → Divine Ruler → Demigod → Ancient God → Supreme God → Eternal God → **Immortal** *(command only)*

## Building
Push to `main` branch — GitHub Actions builds the `.jar` automatically.
